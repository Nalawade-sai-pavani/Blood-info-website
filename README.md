# 🩸 Blood Info Website

> A simple and responsive Blood Information Website built using HTML, CSS, and JavaScript to spread awareness about blood groups, blood donation, and compatibility.

---

## 🌟 Features

✅ Attractive and responsive design  
✅ Information about all blood groups (A+, A-, B+, B-, AB+, AB-, O+, O-)  
✅ Blood donation awareness  
✅ Blood compatibility chart  
✅ Interactive navigation  
✅ Mobile-friendly layout

---

## 🛠️ Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript
- 💻 Visual Studio Code
- 🐙 Git & GitHub

3. Open `index.html` in your browser.
